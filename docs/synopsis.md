# 简介

chrome插件开发

## 什么是Chrmoe插件
Chrome 扩展程序可通过自定义界面、观察浏览器事件和修改网络来提升浏览体验。访问[Chrome 应用商店](https://chromewebstore.google.com/)，查看更多示例，了解扩展程序的用途。

## 需要什么技术
您可以使用与创建 Web 应用相同的 Web 技术来构建扩展程序：HTML、CSS 和 JavaScript。

## 他们可以做什么？
除了 Web API 之外，扩展程序还可以利用 Chrome Extension API 完成不同的任务。如需详细了解概览，请参阅[开发指南](https://developer.chrome.com/docs/extensions/develop?hl=zh-cn)。
